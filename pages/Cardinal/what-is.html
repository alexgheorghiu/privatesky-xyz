Simply put CardinalJS (or simply Cardinal) is a collection of enhanced HTML tags that allows you to build web application.

One example is the very page you see - which was made using CardinalJS.

Cardinal appeared because of the frustration experienced with other web frameworks and it aims to be:

  • a safe guide for programming of SSApp graphic interface
  • a RAD - Rapid Application Development
  • an exensible way of adding new behaviours for web application
